GIT JOINING GUIDE
=================
Registering Public key
                1.  $ ssh-keygen -t rsa  (If you already have a pub key, do not run this command)
                2.  $ cat ~/.ssh/id_rsa.pub  (copy contents)
                3. log on to gerrit site   http://lgsigit.lge.com:8512/ConvAI
                4. Settings -> SSH Keys -> Add Key -> Paste your publick key -> Add   
 
Registering User Information(once)
                $ git config --global user.name vanitha.alagarsamy
                $ git config --global user.email vanitha.alagarsamy@lge.com
                $ git config --global merge.tool kdiff3
 
Public Key Check
                $ ssh -p 29512 lgsigit.lge.com -o "StrictHostKeyChecking no"
                ( "XXXX closed."  is OK result. )


Gererating Workspace folder
                $ mkdir  test

Getting Workspace information from remote server.
                $ cd test

                $git clone ssh://youruserid@lgsigit.lge.com:29512/ConvAI
